# 競合サイト分析

## 傾向
- 画像部分以外は無彩色のみのデザインが多い（配色によるイメージで掲載している本の内容に影響を与えないようにする意図があると思われる）
- レイアウトはオーソドックスで奇をてらわない
- コンバージョンが実店舗に来てもらうことのため、ノストスブックスのフッター部分で「アクセス」の周りに余白をしっかりとって情報が埋もれないようにしている
